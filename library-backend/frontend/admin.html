Admin Guidelines

  • 1Verify bookings only after confirming the student is physically present at the seat.
  • 2Unverified bookings expire automatically after 30 minutes(no manual removal needed.)
  • 3Future date bookings must be reviewed and approved before the booking date.
  • 4Remove bookings only if a student has violated library rules or donnot appear on time.
  • 5Students are limited to 2 bookings per day; do not manually override this limit.
  • 6Check statistics regularly to monitor seat usage and availability.
  • 7Log out of the admin portal when leaving the workstation unattended.
  • 8Report any system issues or booking anomalies to the library management team.
Admin actions are permanent. Removed bookings cannot be restored.
